# 查看监控信息

InfluxDB 控制台提供了丰富的性能监控数据，您可以随时查看实例的运行状态。

## 操作步骤

1. 登录 [InfluxDB控制台](http://tsds-console.jdcloud.com/list)。

2. 在“实例列表”页面，选择目标实例，点击 **实例名称** ，进入实例详情页面。

3. 在”实例详情“页面，点击 **监控**，查看监控信息。

   您可以快速选择查看1小时至14天的数据，也可以输入日期范围查看，时间范围最长可选择1个月。
