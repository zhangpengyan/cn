# 核心概念
以下是区块链数据服务 BDS 帮助文档中使用到的专业术语及其解释，请参考。

|专业术语|解释|
|:--|:-:|
| 链上数据 | 区块链项目的链上交易记录 |
| SQL 语句 | 结构化查询语言，是一种特殊目的的编程语言，是一种数据库查询和程序设计语言 |
| coinbase | 铸币交易的输入部分 |
| 数据表结构 | 由表名、表中的字段和表的记录三个部分组成的 |
| 原生查询 | 使用SQL语句进行查询 |
| Timestamp | 时间戳 |
